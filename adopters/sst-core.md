[SST-Core](https://semanticstep.com/) is a Golang toolkit for the SemanticWeb standards RDF/S, OWL, Turtle and TriG that comes with remote repository support and integrated revision control, all under [PolyForm NonCommercial](/licenses/noncommercial/1.0.0). SST-Core is optimized for cloud applications with special support of industrial product data standards ISO 10303, ISO 15926, ISO 80000, IEC 81346, IEC Common Data Dictionary and others.
