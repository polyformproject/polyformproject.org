This repository contains source files for <https://polyformproject.org>.

## Add Your Project to the Adopters Showcase

Feel free to send GitHub pull requests adding your projects to the adopters showcase.  To do so:

1. Pick a lower-case, alphanumeric slug for your project, like `acme-framework` for "ACME Framework".
2. Create a new Markdown file `./adopters/acme-framework.md`.
3. Add a suitably sized PNG image of your logo, if you have one, as `./adopters/acme-framework.png`.
